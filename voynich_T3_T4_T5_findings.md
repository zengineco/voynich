### Critical 6-grid result for d-stem

d-family DOES take -ar/-al heavily (dar=300, dal=231) but NOT the -edy/-eedy paradigm (dedy=2, deedy=5, deey=6). This rules out d as a process-content stem. d-family is **grammatical morphology**, not lexical content morphology.

### Status

**LOCKED-LITE: d-family is the grammatical-particle stem class.**
- Proclitic subclass: dor, dol, dair (preposition-like, content-noun-leading)
- Enclitic subclass: dy, dal (clause-terminating)
- Coordinator subclass: dar
- Content subclass: daiin (frequent copular/verbal element)

This refines the May-12 "d-stem maximally productive in A-axis" reading: d isn't a content stem inflecting on A-axis; it's the **head of the function-word family** that uses A-axis paradigm slots (-ar/-al) and -aiin/-ain.

### Stem-class typology emerging

**Pattern 1 — "ch/sh open base":**
- High -y (147, 89), high -dy (138, 43), -edy DOMINANT (501, 431), low -eedy
- Takes simple suffixes (-y/-dy) AND process suffix -edy heavily
- chedy is the most common chedy-pattern element corpus-wide

**Pattern 2 — "che/she closed-e base":**
- High -dy (501, 431), LOW -edy (58, 83), very low -eedy
- These stems already end in -e, so -dy directly attaches; -edy is rare (would double the e)
- **che is essentially the same paradigm as ch but with -e built in**

**Pattern 3 — "qok/ot/ok process roots":**
- Low -y, near-zero -dy
- BALANCED -edy / -eedy / -eey (e.g. qok: 269 / 303 / 307 — all three roughly equal)
- These are the **process-register cores** described in May-11 finding
- qokeey + qokeedy = the canonical process-marker pattern

**Pattern 4 — "qoke/oke/ote middle-class":**
- -dy/-edy dominant (qoke: 269/303), low -eey/-eedy
- Use the simple paradigm, not the doubled

**Pattern 5 — "d family":**
- d-edy = 2, d-eedy = 5, d-eey = 6
- **d is OUTSIDE the -edy/-eey paradigm entirely.**
- Confirms d is a function-word stem, not a content stem.

**The clean register axis: -edy ↔ -eey**
- **-edy = process / theoretical pole** (balneo, stars, herbal-A)
- **-eey = naming / recipe / stative pole** (pharm-herbal, cosmo/zodiac)
- -eedy is intermediate (extension/intensifier)

balneo and pharm-herbal are on OPPOSITE poles. balneo edy/eey = 4.04; pharm eey/edy = 9.7. **This is a register continuum, not a binary contrast.**

### Conclusion (T5)

The **-edy/-eedy/-eey** system is **not** a binary pair like -or/-ol. It's a **3-form aspect/register grid**:
- **-edy** = process verb-like, line-late, balneo pole
- **-eedy** = intermediate (intensifier or aspectual extension)
- **-eey** = stative/naming form, pharm/cosmo pole

These three forms partition by **section register**, not by paired contrariety. -edy ↔ -eey is the strongest contrast (edy:eey ratios swing 4.04 → 0.10 across sections).

**Status**: LOCKED. The 3-form -edy/-eedy/-eey grid is a register-aspect inflectional system, distinct from the -or/-ol quality-grade system. Each major content stem (ch, sh, qok, ot, ok) takes all three forms; the proportions differ by stem-class and section.

## §SYNTHESIS — Voynichese inflectional architecture (as of this session)

The Voynich content-word system has at minimum FOUR independent inflectional axes:

1. **-ey/-y (e-grade)**: process vs nominal contrast; corpus-wide constant
2. **-or/-ol (o-grade quality)**: substance-quality contrariety; specific to ch/sh humor pairs and substance class stems
3. **-edy/-eedy/-eey (aspect/register)**: process/intensive/stative; partitions sections
4. **-ar/-al + -aiin/-ain (clitic system)**: grammatical particles, especially d-family

These axes are PARTLY ORTHOGONAL. A given stem typically uses 2-3 axes heavily, with characteristic profile.

**Stem typology (consolidated)**:

| Class | Members | Axes used | Function |
|---|---|---|---|
| Open base | ch, sh | -ey/-y, -or/-ol, -edy, mild -ar/-al | Content nouns + humor-quality grades |
| Closed-e | che, she, oke, ote, qoke | -dy, mild -ey, -or/-ol | Content nouns (e-stem variants) |
| Process root | qok, ot, ok | -edy, -eedy, -eey, -ar/-al, mild -or/-ol | Process/regimen verbal class |
| Substance prefix | cth, kch, dch, tch, otch | Same as base + cth-prefix | Substance-marked content (mostly herbal-A) |
| Function word | d | -aiin/-ain, -ar/-al, -y/-dy | Particles, clitics, copula |
| Discourse | q (only as qol), bare ol | Single word, no productive paradigm | Section-specific discourse markers |

This is the most articulated grammatical picture the project has produced.
